# 📚 Student Data Organizer

The Student Data Organizer is a Python-based command-line application designed to manage student records efficiently using fundamental data structures. It allows users to add, display, update, and delete student information, including ID, name, age, grade, date of birth, and enrolled subjects. The program utilizes lists to store multiple student records, dictionaries to structure individual student data, tuples for immutable attributes like ID and date of birth, and sets to maintain a collection of unique subjects. Through a menu-driven interface, the system ensures easy interaction and demonstrates core programming concepts such as loops, conditionals, and data handling, making it suitable for beginners learning Python and basic data management techniques.
